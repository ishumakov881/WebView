# Лог ошибок

## 17.01.2024

### Ошибка: Несовместимая версия Kotlin
- Описание: Class 'kotlin.Unit' was compiled with an incompatible version of Kotlin
- Файл: Theme.kt, PropertyCard.kt
- Решение: Нужно обновить версию Kotlin в проекте и синхронизировать зависимости
- Как избежать: Всегда указывать точные версии зависимостей в build.gradle

### Ошибка: Отсутствует библиотека Coil
- Описание: Unresolved reference: coil
- Файл: PropertyCard.kt
- Решение: Добавить зависимость io.coil-kt:coil-compose в build.gradle
- Как избежать: Проверять все необходимые зависимости перед использованием компонентов

### Советы по избеганию ошибок:
1. Всегда проверять совместимость версий библиотек
2. Использовать стабильные версии зависимостей
3. Добавлять все необходимые зависимости в начале разработки
4. Тестировать компоненты на наличие всех необходимых библиотек
